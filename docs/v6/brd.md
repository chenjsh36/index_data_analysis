# BRD — 指数数据与信号展示（v6）

## 需求概述

1. **定时跑信号与推送**  
   参考同 workspace 下的 **daily_stock_analysis** 项目（相对路径：`daily_stock_analysis`），在项目部署到 GitHub 后，能够定时每天运行跑信号命令，将结果发送给相关渠道。  
   **已确认渠道**：先支持**邮件**、**钉钉**两种。

2. **静态服务与页面展示**  
   在项目内启动一个静态服务，通过页面展示指数相关数据：  
   - 获取最近 5 年的纳斯达克 QQQ 指数数据，展示曲线走势图  
   - 展示当日运行信号后的相关结果（收盘价、EMA、波动率、推导逻辑、操作建议等）  
   **已确认**：本仓库可启动的静态站点；数据更新按「定时每日跑信号并产出结果，静态页展示最近一次结果与 5 年数据」。
