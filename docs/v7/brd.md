# 一、趋势确认类（用来判断：是不是真牛市）
## 1. 200 日简单移动均线（SMA200）
- 和 EMA 互补：**SMA 更稳、更代表长期真实成本**
- 纳指判断：
  - 价格在 SMA200 上方 = 长期牛市环境
  - 价格在 SMA200 下方 = 熊市/危险区
- 为什么适合纳指：
  纳指长牛，但一旦跌破 SMA200，通常会有**中级调整**，非常准。
- 实战用法：
  **只在 SMA200 上方才考虑做多**，下方一律空仓。

## 2. 布林带中轨（BB Middle Band，默认 20 日）
- 中轨 = 20 日均线
- 判断：
  - 中轨**向上** + 价格在中轨之上 = 中期上升趋势
  - 中轨走平 = 震荡，不做
- 适合纳指：过滤震荡市，只做趋势段。

# 二、趋势强弱类（用来判断：涨得有没有劲，是不是假反弹）
## 3. ADX 平均趋向指数（14 日）
**最被低估、但对纳指极好用的趋势指标**
- 不判断涨跌，只判断：**有没有趋势、趋势强不强**
- 纳指标准：
  - ADX ＜ 20 = 无趋势（震荡，容易亏）
  - ADX 20～25 = 弱趋势
  - ADX ＞ 25 = **真正强趋势**
  - ADX ＞ 30 = 纳指主升浪确认
- 为什么适合你：
  你之前策略跑输，就是因为**在震荡里反复买卖**。
  ADX 能直接把震荡市屏蔽掉。
- 实战用法：
  **只有 ADX > 25 才进场**，胜率立刻提升一大截。

## 4. MACD（12,26,9）
- 看两点就够：
  1. **MACD 线在 0 轴上方** = 多头趋势
  2. 0 轴上方金叉 = **强势加仓点**
- 纳指特点：
  真正主升浪时，MACD 会**长期贴在 0 轴上方**，很少死叉。
- 用法：
  0 轴下方 → 不做多；
  0 轴上方 → 放心持有。

# 三、趋势健康度（判断：上涨是不是健康、会不会崩）
## 5. RSI 相对强弱指数（14 日）
- 对纳指最实用的判断：
  - RSI ＞ 50 = 多头健康趋势
  - RSI 50～70 = **完美强势上涨**
  - RSI ＜ 50 = 走弱
  - RSI ＞ 70 = 过热（随时可能回调）
- 实战意义：
  纳指健康上涨 = **EMA80/200 多头 + RSI 维持在 50～70**
  一旦跌破 50，就是预警。

## 6. 均线多头排列（5/20/60/200）
- 短期均线 > 中期 > 长期
- 完全多头排列 = **纳指最舒服、最安全、最暴利的阶段**
- 你现在的 80/200 就是它的简化版。

---

# 四、熊市/暴跌预警类（纳指最关键：避大跌 = 跑赢基准）
## 7. VIX 恐慌指数（纳指的“天气预报”）
- 和纳指 **完全反向**
- VIX ＜ 20 = 低波动，健康上涨
- VIX 20～30 = 警惕
- VIX ＞ 30 = 恐慌暴跌，必须空仓
- 实战：
  **VIX > 30 时，任何买入信号都忽略**，能避开纳指 90% 的暴跌。

## 8. 20 日收益率波动率（STD）
- 波动率低 = 趋势稳
- 波动率爆高 = 趋势断裂、多空厮杀
- 你现有策略里已经用到，是纳指策略的**必备过滤器**。
